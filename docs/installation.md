# Installation Guide

## Prerequisites
- Node.js (>=14.x)
- npm (>=6.x)
- Git

## Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/nathanaelnienaber/tmrw.it-techdocs.git
   cd tmrw.it-techdocs
   ```
2. Install dependencies (if applicable):
   ```bash
   npm install
   ```
3. You're ready to explore the documentation!
